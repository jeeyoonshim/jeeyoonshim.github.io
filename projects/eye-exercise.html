Eye-Exercise

Exercise in using CSS to draw shapes, and JavaScript to animate it.

Type Web Application

Description The Mouse-Tracking Eyeball Animation is a creative exercise in drawing and animating movement using HTML, CSS, and JavaScript. This project serves as a valuable learning experience, enhancing proficiency in CSS drawing techniques and refining skills in calling elements while manipulating the Document Object Model (DOM).

Languages Used HTML, CSS, JavaScript

Usage Hover the mouse over the screen to witness the dynamic movement of the animated eyeball. The eyeball will seamlessly track the mouse's movement, creating an image with a pair of rolling eyes.

Key Features Utilizes HTML, CSS, and JavaScript for drawing and animating movement.

Learnings This project provided hands-on experience in:
-Drawing and animating elements using a combination of CSS and JavaScript
-Working with and manipulating elements in the Document Object Model (DOM)
