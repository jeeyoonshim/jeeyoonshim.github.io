Bad-Bank

A fullstack banking application

Bad Bank is a fullstack banking application using JavaScript React for the frontend and MongoDB and Express for the backend. The application was made through several iterations and the MongoDB and Express backends changed to BAaS with it hosting on firebase. Bad Bank was a crucial learning experience from designing the frontend UI layout to making it all work and connected with React, and storing the user's data, login credientials and updating the database info accordingly.
