Bad-Bank

A fullstack banking application.

Type Fullstack Web Application

Description Bad Bank is a comprehensive fullstack banking application crafted using JavaScript and React for the frontend, and MongoDB and Express initially for the backend. Through iterative development, the backend transitioned to Backend as a Service (BaaS) hosted on Firebase. The project serves as a pivotal learning experience, encompassing everything from designing the frontend UI layout to establishing seamless connections with React. Bad Bank efficiently manages user data, login credentials, and dynamically updates the database information.

Languages Used Frontend: JavaScript, React | Backend (Initial): MongoDB, Express | Backend (Final): Firebase (BaaS)

Usage Navigate through the user-friendly frontend interface to perform banking operations such as account creation, login, and transaction history.

Key Features Evolved from MongoDB and Express backend to Firebase BaaS for enhanced scalability. Demonstrates the evolution of the project through iterative development, and highlights the frontend design and backend connectivity using React.

Learnings Bad Bank encapsulates invaluable lessons in:
-Fullstack development using JavaScript and React
-Transitioning from traditional backend technologies to BaaS
-Designing and connecting frontend and backend components for a cohesive user experience
