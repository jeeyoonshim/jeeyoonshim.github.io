PacMen-Factory

Using JavaScript

Type Web Application

Description The PacMan Movement Game is a dynamic exercise in utilizing JavaScript and the Document Object Model (DOM) to load, move, and manipulate PacMan images on the screen. This project serves as a hands-on exploration of calling images dynamically, implementing movement, and handling edge detection to seamlessly flip PacMan images when changing direction.

Languages Used HTML, CSS, JavaScript

Usage Click the "Add PacMan" button to introduce a new PacMan onto the screen, or "Add 50 PacMan" to add fifty at once. Click "Start Game" to make all the PacMans move, and "Clear" to remove them all.

Key Features Utilizes JavaScript and DOM to dynamically load and move PacMan images. Implements edge detection to flip PacMan images when changing direction.

Learnings This project provided hands-on experience in:
-Dynamically calling and manipulating images using JavaScript and DOM
-Implementing movement and direction changes for an interactive game
