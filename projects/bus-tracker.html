Bus-Tracker

Tracking routes and marking stops using Mapbox API to fetch data.

Type Web Application

Description Bus Tracker is a project focused on animating map markers to traverse along bus stops. This exercise provided an immersive opportunity to work with the Mapbox API, serving as a challenging venture into retrieving data from APIs and harnessing Mapbox's extensive library of features.

Languages Used JavaScript, Mapbox API

Usage Click on the "Show stops between MIT and Harvard" button on the top left corner and the marker will move along the route between MIT an Harvard, showing the stops.

Key Features Utilizes the Mapbox API for dynamic map rendering and animation, and represents the initial exploration and utilization of APIs in a web application.

Learnings This project is a lesson in:
-Integrating and animating map markers using the Mapbox API
-Fetching and handling data from external APIs
