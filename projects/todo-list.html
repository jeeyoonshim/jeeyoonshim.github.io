Todo-List

A Todo List Application, testing styles in JSX.
Launch / Github

Type Web Application

Description Todo List is a dynamic list application built with JavaScript and React, primarily designed for testing various styles. The application showcases a user-friendly interface displaying tasks along with their respective timestamps. Users can effortlessly add new tasks by clicking on the input box, typing the task details, and hitting enter. To remove an item from the list, simply click on the respective task.

Languages Used JavaScript, React

Usage Click and type into the input box, and hit enter to add new item to the list. Click on the item on the list and it will be deleted.

Key Features Utilizes JavaScript and React for a responsive and interactive user interface. Displays task timestamps to enhance task management.

Learnings This project provided hands-on experience in:
-Testing and implementing different styles within a React application
-Creating a user-friendly task management interface with timestamp functionality
-Enhancing user interactions through dynamic task addition and deletion features
