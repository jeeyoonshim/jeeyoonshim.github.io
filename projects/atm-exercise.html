ATM-Exercise

Simple ATM with Withdraw and Deposit functions.
Launch / Github

Type Web Application

Description ATM Exercise is a simple representation of a banking app developed using JSX and JavaScript with React's useState. The project allows users to perform basic banking operations such as deposit and withdrawal by entering and submitting numerical values. The Account Balance dynamically updates based on the user's actions.

Languages Used JSX, JavaScript (React)

Usage Enter the desired amount for deposit or withdrawal. Submit the form to update the Account Balance.

Key Features Utilizes React's useState for efficient state management. Demonstrates the integration of JSX and JavaScript for interactive user interfaces.

Learnings This project provided hands-on experience in:
-Implementing state management with React
-Developing a simple yet functional web application using JSX
